# shopping-mall-promotions

## Environments
- node 12.x 
- yarn (https://classic.yarnpkg.com/en/docs/install#mac-stable)

## Getting Started

### Installation

```sh
$ yarn

```


### Usage

### Start app on dev mode

```sh
yarn dev:weapp

```

### Build app

```sh
yarn build:weapp

```
